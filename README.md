# ✨ Key Features

## 🎮 Interactive 3D Project Showcase
- GPU-accelerated 3D coverflow carousel using `Swiper.js`
- Smooth project transitions with immersive animations
- Interactive portfolio browsing experience

## ⚡ Event-Driven Frontend Architecture
- Custom browser event dispatchers connecting Skills and Projects sections
- Clicking a skill dynamically auto-scrolls to related projects
- Intelligent UI synchronization for smooth navigation

## 📊 GitHub-Style Analytics
- Dynamically calculated technology usage bars
- Horizontal segmented analytics inspired by GitHub language stats
- Real-time visual representation of project tech stacks

## 🎬 High-Performance Animations
- Advanced motion effects using `Framer Motion`
- Spring physics and staggered animations
- Tooltip bridging and animated layout transitions
- Optimized rendering performance

## 🎨 Premium Agency-Grade UI/UX
- Deep-space dark mode theme
- Metallic gradient typography
- Infinite conic-gradient animated borders
- Custom webkit scrollbars
- Fully responsive across all devices

## 📬 Functional Contact System
- Integrated contact form handling
- Formspree / SMTP backend integration
- Clean user communication workflow

# 🚀 Getting Started

Follow these steps to run the portfolio locally.

---

# 📌 Prerequisites

Install the following:
- Node.js
- npm

Download Node.js:
https://nodejs.org/

---

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/swap821/my-portfolio.git
```

---

## 2️⃣ Navigate to Frontend Directory

```bash
cd my-portfolio/portfolio-frontend
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Start Development Server

```bash
npm run dev
```

---

# 🌍 Run the Application

Visit:

```bash
http://localhost:5173
```
